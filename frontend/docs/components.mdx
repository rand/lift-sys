# Component Library

## Button
- **Props**: `variant`, `onClick`, standard button props.
- **Usage**:
```tsx
<Button variant="default">Click me</Button>
```

## Card
- **Props**: `title` optional, children.
- **Usage**:
```tsx
<Card title="Example">Content</Card>
```

## Badge
- **Props**: `variant` (info|success|warning|error).
- **Usage**:
```tsx
<Badge variant="success">Verified</Badge>
```
